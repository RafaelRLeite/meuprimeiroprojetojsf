# meuprimeiroprojetojsf

Curso Java Web - Full Stack

Este projeto tem o objetivo de criação de um Sistema de cadastro de cliente com formulários e lançamentos utilizando o framework JSF com JPA e Hibernate, focado em funcionalidades

Utilizarei varios recursos do JSF como: caixa de compos, trabalho com datas, multiplos selects, campo de senha e login, autenticação de usuário, radio buttons, command buttons, multiplas seleções, gravar, editar e excluir os dados, lancamentos financeiro, origem e destimo da Nota Fiscal, dentre outros.
